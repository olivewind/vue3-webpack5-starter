# Vue3 Webpack5 Starter

A starter template for Vue3 and Webpack5.

- [x] Vue 3
- [x] Vuex
- [x] Vue Router
- [x] TypeScript 4
- [x] Webpack 5
- [x] Less
- [x] Postcss
- [x] ESlint
- [x] StyleLint
- [x] Prettier
- [x] Docker

## Development

  ``` bash
  yarn

  # then

  yarn dev
  ```

## Build 

  ``` bash
  yarn build
  ```
