# Vue3 Webpack5 Starter

A starter template for Vue3 and Webpack5.

- [x] vue3
- [x] vuex
- [x] vue router
- [x] typescript 4
- [x] webpack 5
- [x] less
- [x] postcss
- [x] eslint
- [x] stylelint
- [x] prettier
- [x] docker

## Development

  ``` bash
  yarn

  # then

  yarn dev
  ```

## Build 

  ``` bash
  yarn build
  ```
